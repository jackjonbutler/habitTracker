## 🚀 Quick Start (3 Steps)

### 1. Open the Backend Folder
```bash
cd backend
```

### 2. Read the Setup Guide
Open `backend/QUICK_START.md` - it has everything you need to:
- Install dependencies
- Set up credentials
- Start the server
- Test the API

### 3. Start Coding!
```bash
npm install
cp .env.example .env
# Edit .env with your credentials
npm run dev
```
